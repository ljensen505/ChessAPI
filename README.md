# Chess API

This is a RESTish API meant for my ChessPVP game, though it could be easily used for any simple Chess application.

Running this is straightforward:
<li><code>clone</code> this repo</li>
<li>Within the project directory, create a virtual environment with <code>python3 -m venv venv</code> or similar</li>
<li>activate the venv with <code>source venv/bin/activate</code> or similar</li>
<li>Create a .env file to specify a port number. This file should contain one line of <code>PORT=8000</code> where '8000' is whichever port you'd like to use</li>
<li><code>python main.py</code></li>
